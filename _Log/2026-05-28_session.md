### 실패 대응

| 증상 | 원인 | 해결 |
|---|---|---|
| `gpg: signing failed: Inappropriate ioctl for device` | GPG agent에 TTY 정보 없음 | `export GPG_TTY=$(tty)` 후 재시도. `.zshrc`에 영구 등록 권장. |
| pre-commit hook block | high-entropy 매치 | 해당 라인 확인 → password면 마스킹 → false positive (PGP key 등)이면 commit message에 사유 명시 후 user 판단으로 `--no-verify` |
| `commit -S` 후 GUI에서 unverified | local GPG key가 GitHub 미등록 | https://github.com/settings/keys 에서 public key 등록 (E81313B5B651B0D9) |
| `Permission denied (publickey)` on push | SSH key 미로드 | `ssh-add ~/.ssh/id_ed25519` (또는 해당 key path). macOS는 `ssh-add --apple-use-keychain`로 keychain 저장. |
